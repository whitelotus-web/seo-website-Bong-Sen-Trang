# Website Bông Sen Trắng

## Hướng SEO

- Kênh chính: Google Search.
- Đo lường chính: Google Search Console của website Bông Sen Trắng.
- Không dùng Bing Webmaster Tools để đánh giá tiến độ SEO của dự án này.

Website tĩnh cho dịch vụ sản xuất, thi công lắp đặt biển quảng cáo tại Hà Nội.

## File chính

- `index.html`: trang chủ/landing page SEO local.
- `assets/css/styles.css`: giao diện responsive.
- `assets/js/main.js`: menu di động và năm ở footer.
- `robots.txt`, `sitemap.xml`: chuẩn bị cho Google index.
- `local-server.js`: preview local bằng `npm run preview`.

## Việc cần làm trước khi public

1. URL miễn phí đang chạy: `https://lam-bien-quang-cao-bong-sen-trang.netlify.app/`.
2. Deploy lại Cloudflare Pages khi sửa nội dung bằng lệnh trong `DEPLOY-MIEN-PHI.md`.
3. Xác minh website trong Google Search Console.
4. Tạo hoặc cập nhật Google Business Profile với đúng tên, địa chỉ, số điện thoại.
5. Xin review thật từ khách cũ, ưu tiên review có ảnh công trình.
